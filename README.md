# Number-Place 數獨網頁版

經典 9x9 數獨遊戲，單一 HTML 檔案，無需任何框架或依賴。

## 功能特點

- **三種難度**：簡單、中等、困難
- **視覺輔助**：選取高亮、同數字標示、行列表標示
- **即時驗證**：衝突數字紅色警示
- **輔助功能**：候選數顯示、提示、重置
- **鍵盤操作**：支援方向鍵移動、數字鍵輸入
- **響應式設計**：支援桌面與行動裝置

## 使用方式

直接用瀏覽器開啟 `sudoku.html` 即可開始遊戲。

```bash
# 或者啟動本地伺服器
python3 -m http.server 8000
# 然後訪問 http://localhost:8000/sudoku.html
```
